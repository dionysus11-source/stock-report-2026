## 메모
### 1단계: 주봉 체크 (큰 흐름 - "지금 안전한가?")

주봉은 전체적인 '방향성'을 필터링합니다.

- [ ] **가격과 20주선의 관계:** 현재 주가가 **20주선 위**에 있는가?
    
    - _Yes:_ 대세 상승장입니다. 1차 진입이 매우 안전합니다.
        
    - _No:_ 역배열 혹은 바닥권입니다. 반등이 나와도 20주선에 맞고 다시 떨어질 수 있으니 주의가 필요합니다.
        
- [ ] **20주선의 기울기:** 20주선이 **우상향 혹은 수평**을 유지하고 있는가?
    
    - _Yes:_ 추세가 살아있습니다.
        
    - _No:_ 하향 곡선이라면 주가가 일시적으로 올라도 다시 밀릴 힘이 강합니다.
        

### 2단계: 일봉 체크 (상세 타이밍 - "오늘 사도 되는가?")

일봉은 실제 '매수 버튼'을 누르는 타이밍을 결정합니다.

- [ ] **주가와 20일선의 관계:** 현재 주가가 **20일선(한 달 평균) 위**에 안착했는가?
    
    - _Yes:_ 단기적인 매수 심리가 회복되었습니다. 진입 적기입니다.
        
    - _No:_ 아직 하락 압력이 큽니다. 20일선을 돌파할 때까지 기다리는 것이 좋습니다.
        
- [ ] **5일선과 20일선의 관계 (골든크로스):** **5일선이 20일선 위**로 올라왔는가?
    
    - _Yes:_ 강력한 매수 신호입니다. 단기 반등 탄력이 붙은 상태입니다.
        
    - _No:_ 5일선이 아래에 있다면(데드크로스 상태), 주가가 20일선 근처에서 저항을 받을 수 있습니다.
        
- [ ] **캔들 확인:** 오늘(혹은 전일) 캔들이 **양봉(빨간색)**으로 마감했는가?
    
    - _Yes:_ 실제 매수세가 들어오고 있음을 의미합니다.

## 생각 (질문)
| **체크 합계**       | **판정 결과**        | **행동 지침**                                    |
| --------------- | ---------------- | -------------------------------------------- |
| **모두 Yes (5개)** | **적극 매수 (Best)** | **1차 매수 최적기.** 수익이 날 확률이 매우 높습니다.            |
| **4개 Yes**      | **안전 진입 (Good)** | 큰 무리 없는 진입 시점입니다. 전략대로 실행하세요.                |
| **2~3개 Yes**    | **주의 (Caution)** | 주봉이나 일봉 중 하나가 꺾여 있습니다. 소액만 진입하거나 관망하세요.      |
| **0~1개 Yes**    | **관망 (Danger)**  | **절대 금지.** 1차 매수하자마자 지하실(2~7차)로 끌려갈 위험이 큽니다. |
|                 |                  |                                              |
